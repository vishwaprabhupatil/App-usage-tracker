# App Usage Tracker

A parental monitoring and screen time tracking application designed to help parents understand and control their child’s smartphone usage in a healthy and responsible way.

---

## 🚀 Features

- 📊 Tracks total screen time accurately  
- 📱 Displays individual app usage statistics  
- 👨‍👩‍👧 Parent dashboard to monitor child activity  
- ⛔ Ability for parents to block selected apps  
- 🔔 Background service with notification support with auto syncing every 30 mins
- 🔐 Secure access and permission-based tracking  

---

## 🛠️ Tech Stack

- **Frontend:** Flutter (Dart)
- **Backend:** Java (Android Services)  
- **Database:** Firebase  
- **Platform:** Android  

---

## ⚙️ How It Works

1. The child installs the app and grants usage access permissions  
2. App usage data is collected using Android Usage Stats APIs  
3. Screen time data is synced and displayed to the parent  
4. Parents can view usage reports and block apps when required  

---

## 🎯 Use Cases

- Parents monitoring children’s phone usage  
- Reducing screen addiction among students  
- Promoting healthy digital habits  
- Educational and hackathon demonstrations
